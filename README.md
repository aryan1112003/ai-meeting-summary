## 🎯 How It Works

<div align="center">

\`\`\`mermaid
sequenceDiagram
    participant U as User
    participant A as App
    participant AI as Gemini AI
    U->>A: Upload Recording
    A->>AI: Process Audio
    AI->>A: Generate Summary
    A->>U: Present Results
\`\`\`

</div>

1. 📤 **Upload Your Recording**
   - Drag & drop your file
   - Supports MP3, WAV, MP4
   - Up to 100MB

2. ⚡ **Watch the Magic**
   - Real-time processing
   - Progress tracking
   - Smart analysis

3. 📋 **Get Your Summary**
   - Structured overview
   - Action items
   - Key decisions
   - Next steps

4. 📥 **Export & Share**
   - Download as PDF
   - Share with team
   - Custom formatting
